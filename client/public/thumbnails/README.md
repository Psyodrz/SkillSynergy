# Skill Thumbnails Directory

This directory contains reusable, category-based skill thumbnails.

## Structure

- Each subfolder represents a skill category
- Each category has 2-3 thumbnail variants for visual diversity
- Thumbnails are randomly assigned when skills are created

## Categories

- tech/ - Tech & Development, Data & AI
- design/ - Design & Creative, Arts & Music
- business/ - Business, Career & Job Prep, Personal Finance
- lifestyle/ - Life Skills & Productivity, Health & Fitness, Home & Lifestyle
- language/ - Language & Communication
- hobbies/ - Hobbies & Games
- academics/ - Academics & Exams
- relationships/ - Parenting & Relationships

## Usage

Images are statically served and assigned via the `thumbnail` field in the skills table.
